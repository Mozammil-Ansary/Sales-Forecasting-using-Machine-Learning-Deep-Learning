# Sales-Forecasting-using-Machine-Learning-Deep-Learning
Sales Forecasting using ML &amp; Deep Learning — End-to-end forecasting project using restaurant sales data to predict future sales trends. Includes EDA, feature engineering, regression modeling, XGBoost, LSTM-based time-series forecasting, and business insight generation for data-driven decision making.

## 📈 Sales Forecasting using Machine Learning & Deep Learning

### 📌 Project Overview

Sales forecasting plays a critical role in helping businesses optimize inventory management, staffing, budgeting, and operational planning.

- This project builds an end-to-end forecasting system using restaurant sales data to predict future sales trends through Machine Learning and Deep Learning techniques. The project combines data analysis, feature engineering, regression modeling, and time-series forecasting to generate meaningful business insights and accurate sales predictions.

## 🎯 Business Problem

- Businesses often struggle to accurately predict future sales due to changing customer demand and seasonal trends.
- The objective of this project is to:

   - Forecast future sales trends
   - Identify business growth patterns
   - Support inventory and operational planning
   - Enable data-driven business decisions


📂 Dataset Information

The project uses multiple restaurant-related datasets containing:
- Restaurant details
- Item information
- Historical sales transactions

- Dataset Includes
   - Sales quantity
   - Revenue information
   - Restaurant IDs
   - Item IDs
   - Date & time details

## 🔍 Exploratory Data Analysis (EDA)

Performed comprehensive data analysis to uncover:

- Daily and monthly sales trends
- Seasonal demand patterns
- Restaurant-wise sales performance
- Item popularity analysis
- Revenue growth insights


## 📊 Key Insights
- Certain restaurants generated significantly higher revenue
- Sales varied across weekdays and months
- Some items showed consistently high demand
- Seasonal patterns influenced overall sales performance


## ⚙️ Data Preprocessing & Feature Engineering

The following preprocessing techniques were applied:

- Merging multiple datasets
- Handling missing values
- Date-time transformation
- Feature engineering using:Day, Month, Quarter, Year
- Data cleaning and formatting

##🤖 Machine Learning Models Used

Implemented and compared multiple regression models:

- Linear Regression
- Random Forest Regression
- XGBoost Regression


📌 Best Performing Model

- XGBoost achieved the best performance by capturing non-linear sales patterns effectively.

🧠 Deep Learning & Time Series Forecasting

= Implemented LSTM (Long Short-Term Memory) model for time-series forecasting to capture sequential sales dependencies and predict future sales trends.

## 📈 Model Evaluation

The models were evaluated using:

- RMSE (Root Mean Squared Error)
- Forecast trend comparison
- Prediction visualization


## 📊 Results
- XGBoost produced the lowest RMSE
- LSTM successfully captured sequential sales patterns
- Generated future sales forecasts for the next 12 months
